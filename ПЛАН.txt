Создание таблицы с каким-либо данными

1. загрузка данных:
	+ в отдельном модуле, data.js;
	+ с сервера, асинхронно;
	+ использовать плагин axios, либо через XmlHttpRequest;
	+ обрабатывать ошибки;
	+ не допускать зависания;
	+ показывать индикатор загрузки;
	+ загружать данные в переменную let data, доступную во всех модулях, либо в localStorage;

2. html-страница:
	- использовать плагин REDOM для операций с DOM;
	+ для исходных данных использовать операцию updateTable, данные брать из data;
	- при использовании параметров фильтрации/сортировки сперва изменять data через функции filterData, sortData;
	+ выстраивать новую таблицу;
	+ очищать контейнер от старой таблицы, записывать в него через setChildren новую;

3. операции с таблицей:
	+ сортировка - вызывать updateTable с параметром сортировки - именем столбца;
	- фильтрация - вызывать updateTable с параметром фильтрации - введенными текстовыми данными из панели фильтрации;
	+ пагинация:
		+ отображать несколько страниц, максимум 50 записей на страницу;
		+ добавить панель перехода между страницами под/над таблицей;
	- выделение строки целиком - отображать под таблицей панель с подробной информацией о записи;

4. форма добавления новой записи:
	- инпуты для ввода данных;
	- добавлять в data;
	- сразу после добавления отображать первой строчкой (до первой сортировки или фильтрации);
	- после добавления сортировку и фильтрацию сбрасывать;

5. проекты с похожими запросами:
	- CRM SKB - постройка таблицы, сортировка, асинхронная загрузка данных;
	- dpo_javascript_basic/11-client-server - простейшая пагинация при загрузке данных из сети;
	- ПРАКТИКА/База студентов - постройка таблицы, сортировка, работа с localStorage, форма добавления записи;

6. способы выполнения:
	- с созданием таблицы как отдельного компонента, формирование которого будет производиться вызовом js-функции и возвратом объекта, в объекте будет содержаться DOM-элемент, функция апдейта DOM-контейнера с таблицей (по параметру), функции формирования DOM с параметром json (данные для отображения);
	- отдельный простой проект со статичным index.html и отдельной функцией для апдейта контейнера с таблицей, плюс модуль с api-функциями загрузки данных с сервера.
	- отдельный простой проект со статичным index.html, но с подключением объекта с таблицей как отдельного компонента.
